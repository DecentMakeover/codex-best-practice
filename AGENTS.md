# AGENTS.md

## Repo purpose

This repository is a README-first Codex handbook. The repo should demonstrate
real Codex patterns, not just describe them.

## Working rules

- Keep `README.md` and `index.html` aligned at the section level.
- Prefer official OpenAI Codex docs when making claims about features or config.
- If you add a concept to the README, point at a real file in the repo when possible.
- Avoid marketing copy. Prefer dense, example-driven explanations.
- Keep example files small enough to copy into real projects.

## Validation

- If you change HTML, serve the repo locally and check the root page.
- If you change links, verify that internal file references and official-doc links still resolve.
- If you change config or skill examples, keep them consistent with the handbook docs.
